O que é o Arduino

É uma placa com uma série de microcomputadores e componentes integrados.

O que é um microcontrolador

É um computador pequeno em um único chip

microcontrolador do Arduino

Modelo: ATmega328P UNO
Fabricante: Atmel

Linguagem de programação do Arduino

C++ e Mycropython

Onde o Arduino é programado

No programa Arduino IDE

Qual é o valor em volts da entrada analógica no arduíno e a resolução em bits

Valor: 0 à 5V
resolução: 0 à 1023

Valor da entrada e saída do Arduino em volts

0: 0 volts
1: 5 volts

O que é um multimetro

É um dispositivo capaz de medir diversas grandezas elétricas

Quais grandezas um multimetro consegue medir

Tensão, corrente e resistência

O que é escala de medição

É a faixa de valores para uma determinada grandeza que o instrumento pode medir com precisão

Qual a lei que o multímetro aplica para realizar as medidas


Se baseia na Lei de Ohm

O que é teste de continuidade e para que serve

É um teste que verifica se há conexão elétrica entre dois pontos de um circuito. Ele serve para prevenir curtos-circuitos

Como é a medição de uma resistência no resistor

É usado um multímetro na escala de ohms, começando em uma escala alta e diminuindo para receber um resultado mais preciso

Como é a medição de uma tensão no circuito com resistor e led

É usado um multímetro na escala de tensão contínua.

Como é a medição de uma corrente elétrica no circuito

Usando um multímetro e conectando a ponta de prova preta no terminal COM e a vermelha na entrada de corrente, posicionando o seletor para a escala correta de corrente (contínua ou alternada) e ligando o circuito.
